## 下载

### Docker

```
docker pull apachecn0/matplotlib-doc-zh
docker run -tid -p <port>:80 apachecn0/matplotlib-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g matplotlib-doc-zh
matplotlib-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
